# Template Selection

Expo Router mobile starter with platform-managed Convex backend and Convex Auth.

Use when:
- The user asks for a mobile, iOS, Android, phone, React Native, or Expo app
- The request needs persistence, realtime shared state, auth, user-owned data, AI-backed data, or backend functions
- The app should share data across users or sessions
- The app needs email/password auth, email OTP verification, password reset, or guest access
- The app needs Expo Go preview compatibility and platform-managed backend provisioning

Avoid when:
- The request is a UI-only mobile prototype with no backend state
- The user asks for a web-only app, Cloudflare Worker app, presentation, or backend workflow
- The request requires custom native modules, dev clients, EAS builds, prebuild, or app-store submission
- The user wants to bring an existing Convex project

Built with:
- Expo Router, React Native, Expo Go-compatible packages, TypeScript, Convex, Convex Auth, platform-managed Convex provisioning
